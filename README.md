# Vesuvius Challenge

Reading the Herculaneum scrolls using machine learning and computer vision.

## Getting Started

1. Clone this repo
2. Set up Python environment:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
3. Download sample data from https://scrollprize.org/data
4. Explore notebooks in `/notebooks/`

## Project Structure

- `data/` — Downloaded scroll data (gitignored)
- `notebooks/` — Exploration and experiment notebooks
- `src/` — Source code
- `models/` — Trained model checkpoints (gitignored)
- `outputs/` — Generated outputs and submissions (gitignored)
